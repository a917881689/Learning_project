Apache 服务器（只能处理静态资源,静态网页,多媒体文件）
Tomcat 服务器 （可以同时处理动态和静态资源）

服务器有什么特点：
    1.尽量不ni机linus (可以20年左右不关机)：虚拟机+Redis
    2.分布式协作
    3.http://localhost:8888/ 默认找到tomcat服务器
    localhost: 表示本机
    8888 代表程序的端口号,

javaEE 
    java Resource/src (java类 () 三层架构)

    WebContent javaWeb工程的默认路径

    所有的页面 html、jsp、
使用Eclipse怎么将web工程发布到tomCat服务器
    1.Eclipse中继承tomCat环境
    2.创建本地server(修改工程的发布路径)
    3.创建动态网页工程Dynamic Web Project(主要勾选web.xml)
    4.在WebContent(web应用的默认路径)中创建一个index.html中
    5.Run as ---- run on server (自动发布到tomCat服务器中)
